1 Project Overview

CryptoPay is a full-stack cryptocurrency recurring payment platform. It enables businesses to accept automated crypto subscriptions from customers using Trust Wallet and MetaMask. The platform consists of a React frontend, Node.js backend, PostgreSQL database, and Solidity smart contracts deployed on EVM-compatible blockchains.

Deployment Target: Vercel Pro Plan — $20/month (Serverless)

3 Vercel Environment Variables

These values must be set in Vercel Dashboard → Project Settings → Environment Variables:

Variable Name Required? Value / How To Generate
DATABASE_URL Must Connection string from your PostgreSQL provider.
postgresql://user:pass@host/dbname?sslmode=require
SESSION_SECRET Must Generate with:
node -e "console.log(require('crypto').randomBytes(32).toString('base64url'))"
ENCRYPTION_MASTER_KEY Must Generate with:
node -e "console.log(require('crypto').randomBytes(32).toString('base64'))"
CRON_SECRET Must Generate with:
node -e "console.log(require('crypto').randomBytes(32).toString('base64url'))"
EXECUTOR_PRIVATE_KEY Should Private key from your Executor MetaMask wallet.
Format: 0x followed by 64 hex characters
DEPLOYER_PRIVATE_KEY Should Private key from your Deployer MetaMask wallet.
Used for wallet-switching & on-chain cancellations.
SEPOLIA_RPC_URL Nice From Alchemy/Infura dashboard. Format:
https://eth-sepolia.g.alchemy.com/v2/YOUR_API_KEY
⚠️ Security: Never share private keys or secrets publicly. All values should be entered only in the Vercel Environment Variables dashboard, never committed to Git.
